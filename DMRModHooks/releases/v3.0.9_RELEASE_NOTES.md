# v3.0.9 Release Notes - GPS Distance Display Enhancements

**Release Date**: March 9, 2026

## ЁЯзн GPS Distance Display Enhancements

This release focuses on improving the GPS distance information display with directional arrows, compass bearings, and dual distance units for better situational awareness.

### тЬи New Features

#### Directional Arrows with Compass Bearings
- **8-direction compass arrows**: тЖСN, тЖЧNE, тЖТE, тЖШSE, тЖУS, тЖЩSW, тЖРW, тЖЦNW
- Shows direction from your location to the channel
- Real-time bearing calculation using GPS coordinates
- Visual orientation at a glance
- 45-degree sectors for accurate direction mapping

#### Dual Distance Display
- **< 1km**: Shows meters only (e.g., `тЖСN 250m`)
- **1-10km**: Shows kilometers (e.g., `тЖТE 5.2km`)
- **> 10km**: Shows both miles and kilometers (e.g., `тЖШSE 96.7mi (155.6km)`)
- Accommodates users familiar with either measurement system
- Smart formatting based on distance range

#### Improved UI Layout
- Widened location text area from 30% to 40% screen width
- Prevents text wrapping on long distance strings
- Better readability at 12sp font size
- Maintains right-alignment for clean appearance
- Changed layout split from 70%/30% to 60%/40% (info/location)

### ЁЯФз Technical Details

- **Bearing Calculation**: Uses haversine formula with atan2 for accurate compass bearings
- **Direction Mapping**: 8 compass directions with 45-degree sector boundaries
- **Distance Formatting**: Automatic unit selection based on distance range
- **No Internet Required**: Uses device GPS/location only
- **Performance**: Real-time updates when switching channels

### ЁЯУж Installation

1. **Requirements**:
   - LSPosed Framework v1.9.2+
   - Magisk v24+ with Zygisk enabled
   - Rooted Ulefone Armor 26 Ultra (or compatible device)

2. **Install Module**:
   ```bash
   adb install -r app-release.apk
   ```

3. **Enable in LSPosed**:
   - Open LSPosed Manager
   - Enable "DMRModHooks" module
   - Select "PriInterPhone" app
   - Reboot device
