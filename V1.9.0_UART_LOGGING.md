# DMRModHooks v1.9.0 - UART Serial Logging Implementation

**Release Date:** February 27, 2026  
**Priority:** #1 from DMR_FIX_ROADMAP.md  
**Status:** ✅ Built and Installed

## Overview

Version 1.9.0 implements comprehensive UART serial communication logging for the /dev/ttyS0 interface between the Android app and the DMR hardware module. This is the foundational step toward firmware reverse engineering and protocol analysis.

## What's New

### 🔌 UART Serial Logging Hooks

**New Methods:**
- `hookSerialCommunication(lpparam)` - Main hook installer for Serial class
- `logUartData(direction, data, binaryLog, textLog)` - Dual-format logging with packet parsing

**Hooked Methods:**
1. `Serial.write(byte[])` - Captures all TX (transmit) data
2. `Serial.read(byte[], int)` - Captures all RX (receive) data

### 📝 Dual-Format Logging

**Binary Log Format** (`.bin` files):
```
[Direction Marker: 1 byte] (0x01=TX, 0x02=RX)
[Timestamp: 8 bytes] (milliseconds since epoch)
[Data Length: 4 bytes] (integer)
[Raw Data: N bytes] (actual packet data)
```

**Text Log Format** (`.txt` files):
```
========================================
TX @ 2026-02-27 22:06:15.123
Length: 163 bytes
----------------------------------------
Hex: 00 01 02 03 04 05 06 07 ...
----------------------------------------
Packet Analysis (DigitalMessage):
  RX Freq: 467700000 Hz (467.7 MHz)
  TX Freq: 467700000 Hz (467.7 MHz)
  Local ID: 4606002
  TX Contact: 16777215
  Contact Type: 2 (ALL) 🔥🔥🔥 MONITORING MODE
  Power: 1
  Color Code: 1
  Slot: 1
  Groups: [10, 17, 32, 99] (4 active)
========================================
```

### 🎯 Real-Time Packet Parsing

Automatically parses DigitalMessage packets (163 bytes) with field extraction:
- **Offset 0-3:** RX Frequency (big-endian int)
- **Offset 4-7:** TX Frequency (big-endian int)
- **Offset 8-11:** Local DMR ID
- **Offset 12-139:** Group List (32×4 bytes)
- **Offset 140-143:** TX Contact (big-endian int)
- **Offset 144:** **Contact Type** (0=PERSON, 1=GROUP, 2=ALL)
- **Offset 145:** Power level
- **Offset 146:** Color Code
- **Offset 147:** Time Slot

### 📂 Log Storage

**Location:** `/sdcard/DMR/uart_logs/`

**Files Generated:**
- `uart_YYYYMMDD_HHMMSS.bin` - Binary log for protocol analysis tools
- `uart_YYYYMMDD_HHMMSS.txt` - Human-readable log for quick inspection

**Permissions:** Ensure storage permissions granted for DMR app

### Hook Architecture

```
com.pri.prizeinterphone.serial.Serial
├── write(byte[]) ← TX Hook (beforeHookedMethod)
│   └── logUartData("TX", data, ...)
│       ├── Append to binary log with metadata
│       ├── Generate hex dump
│       └── Parse DigitalMessage fields
│
└── read(byte[], int) ← RX Hook (afterHookedMethod)
    └── logUartData("RX", data, ...)
        ├── Append to binary log with metadata
        ├── Generate hex dump
        └── Parse DigitalMessage fields
```

### Thread Safety

All file I/O operations run in background threads to avoid blocking the main serial communication:
```java
new Thread(new Runnable() {
    @Override
    public void run() {
        // File operations here
    }
}).start();
```

## Usage Instructions

### 1. Activate in LSPosed

1. Open LSPosed Manager app
2. Navigate to Modules → DMRModHooks
3. **Verify version shows 1.9.0**
4. Ensure "com.pri.prizeinterphone" is checked in target apps
5. Force stop DMR app or reboot device

### 2. Grant Storage Permissions

```bash
adb shell pm grant com.pri.prizeinterphone android.permission.WRITE_EXTERNAL_STORAGE
adb shell pm grant com.pri.prizeinterphone android.permission.READ_EXTERNAL_STORAGE
```

### 3. Start Logging

1. Open DMR radio app
2. Change channels, toggle MON button, make calls
3. Logs automatically saved to `/sdcard/DMR/uart_logs/`

### 4. Retrieve Logs

```bash
adb pull /sdcard/DMR/uart_logs/ C:\DMR_Logs\
```

### 5. Monitor in Real-Time

```bash
adb logcat | Select-String "DMRModHooks"
```

Look for:
```
DMRModHooks: 🔌 Hooking UART serial communication (/dev/ttyS0)...
DMRModHooks: 📝 UART logs will be saved to: /sdcard/DMR/uart_logs
DMRModHooks: ✅ UART serial hooks installed successfully!
DMRModHooks: UART TX 163 bytes
DMRModHooks: UART RX 163 bytes
```
