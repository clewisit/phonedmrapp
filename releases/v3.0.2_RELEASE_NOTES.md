## 🎯 Release Highlights

### Why v3.0.2? (Skipping v2.x)

This release jumps from v1.7.0 → v3.0.2 to deliver working features now while DMR group calling improvements (planned for v2.x) are being investigated separately.

**The Strategy**:
- v1.x: Stable feature releases (export/import, location, transcription)
- v2.x: **IN DEVELOPMENT** - DMR enhancements (group calling fixes)
- v3.x: **CURRENT** - Analog features + branding (ready to use!)

Rather than delay a fully working analog monitoring feature for weeks/months while debugging complex DMR protocol issues, I'm releasing v3.0.2 now with stable analog MON functionality. DMR improvements will come in v2.x when ready.

## 🆕 New Features

### 1. Analog MON (Monitor) Button ⭐

A true scanner-style monitoring button for analog channels:

**What It Does**:
- 📻 **MON ON**: Opens squelch completely (sq=0) - hear ALL audio
- 🔇 **MON OFF**: Restores normal squelch filtering
- 🎨 **Visual**: Orange = monitoring active, Gray = normal
- 🔄 **Smart Reset**: Auto-disables when switching channels
- 📢 **Feedback**: Toast notifications confirm state

**Technical Achievement**:
- Direct hardware messaging (bypasses database)
- Manual AnalogMessage construction
- Preserves all 11 channel parameters
- Handles timing correctly with firmware

**Status by Channel Type**:
- ✅ **Analog Channels**: Fully working and stable
- ⏸️ **Digital Channels**: Button hidden until DMR "All" mode fixed

**Perfect For**:
- Monitoring busy simplex frequencies
- Repeater traffic listening
- Emergency channel monitoring
- Amateur radio scanning

## ⏳ Known Limitations

### DMR Groups "All" Mode

**Issue**: Reception problems when monitoring all talk groups simultaneously

**Status**: 
- 🔬 Under active investigation
- 📅 Will be addressed in v2.x release
- 🚫 MON button disabled for DMR channels until fixed

**Workaround**: Use specific talk group IDs instead of "All" mode

**Why This Matters**: 
This is a complex hardware-level protocol timing issue that requires extensive UART analysis. Rather than delay this release, I'm shipping the working analog MON feature now and continuing DMR investigation separately.

## 📥 Installation

### Method 1: Install Pre-built APK (Recommended)

1. Download `DMRModHooks-v3.0.2.apk` from [Releases](../../releases/tag/v3.0.2)
2. Install via ADB:
   ```bash
   adb install -r DMRModHooks-v3.0.2.apk
   ```
3. Enable in LSPosed:
   - Open LSPosed Manager
   - Find "DMRModHooks"
   - Enable for "PriInterPhone" (com.pri.prizeinterphone)
   - Reboot device

### Method 2: Build from Source

```bash
git clone https://github.com/IIMacGyverII/phonedmrapp.git
cd phonedmrapp/DMRModHooks
./gradlew assembleDebug
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### Verification

After installation and reboot:
1. Open DMR app
2. Look for toast: "✓ DMR Mod Hooks Active! v3.0.2"
3. Check MON button appears on analog channels
4. Go to Device → Information to see MacGyver Mod Version
